# OTT Content Analytics Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on OTT platform data using Python.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Movies vs TV Shows comparison
- Year-wise content growth
- Genre distribution
- Rating analysis

## Key Insights
- Content production increased significantly after 2016.
- Movies dominate the platform.
- Certain genres consistently appear more frequently.
- Mature audience ratings are common.

## Conclusion
The OTT industry shows rapid expansion with a strong focus on movie content and high-engagement genres.
